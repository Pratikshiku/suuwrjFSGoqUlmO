## Cinema Ticket Purchase System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/2f8af2ea0d654e8f936fa8eb624a97ef/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Movies
5. Forum
6. News Information
7. Personal Center
8. Movie Favorites
9. Movie Orders
10. Background Management


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Movie Management, Movie Review Management, Movie Favorites Management, Movie Order Management
5. Basic Data Management (Movie Type Management, News Type Management)
6. Forum Management
7. News Information Management
8. User Management
9. Carousel Information
